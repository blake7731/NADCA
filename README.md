To use this program the user can do the following steps:\
1)Record:They can record for a specified amount of time an it will be save to output.wav\
2)Transcribe:any wav file can be imported but will go to gemini to transcribe\
3)Edit:Previous step required but with inserted prompt can give response based on the transcription\
\
Gemini's API limits overload quickly so only a few uses may be used currently
